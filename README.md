# PriceCheck

**Понимайте реальную цену покупок — не только то, что написано на ценнике.**

PriceCheck помогает ответить на главный вопрос перед кассой: *была ли скидка на самом деле?* Сервис опирается на историю чеков и агрегированные данные: вы видите, **сколько за тот же товар платят другие люди** и **как цена отличается между торговыми сетями**. Это не «ещё один сканер чеков», а инструмент для честного сравнения — для личного бюджета и для решений на уровне бизнеса.

Текущий этап: **MVP / альфа**.

Основной репо скрыт.

---

## Зачем это пользователю

- **Проверка «честной» скидки** — сопоставление вашей цены с тем, что платят другие за сопоставимые позиции.
- **Контекст по сетям** — насколько выгодна покупка относительно других магазинов и форматов.
- **История и аналитика** — чеки (в т.ч. с фото/PDF/текста) превращаются в структурированные данные и отчёты, а не в папку скриншотов.

## Зачем это бизнесу (B2B)

- **Корпоративные и интеграционные сценарии** — API для встраивания в учёт, лояльность, финтех и внутренние панели.
- **Масштабируемый backend** — единая платформа распознавания, хранения и аналитики под ваш продукт или white-label.
- **Админка для операций** — управление пользователями, настройками распознавания, ролями, мониторингом и служебными процессами без правки кода под каждый кейс.

Если вам нужна **демонстрация, пилот или обсуждение лицензирования / партнёрства** — ориентируйтесь на возможности из PRD и корпоративного контура; технический запуск и среда описаны в документации ниже.

---

## Админ-панель (в двух словах)

Веб-интерфейс на стороне сервиса: вход по ролям, работа с пользователями и данными, настройки распознавания чеков и продуктового поиска, инструменты для нормализации товаров, достижений, подписок, резервного копирования и прочих операционных задач. Публичная OpenAPI по умолчанию отключена — это сознательное решение по безопасности; для интеграций используйте согласованные контракты и среду, где документация включена осознанно.

## Стек

| Слой | Технологии |
|------|------------|
| API и UI | FastAPI, шаблоны админки |

| Наблюдаемость | Prometheus, Grafana |
